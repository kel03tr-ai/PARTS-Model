# PARTS Framework Practice Tool

An interactive prompt-building tool for the AIETP 2026 training programme. Educators learn to write effective AI prompts using the PARTS framework: Persona, Aim, Recipients, Theme, Structure.

## What It Does

During Session 1, participants are introduced to the PARTS framework for writing structured prompts to AI tools. This tool lets them practise building, evaluating, and refining prompts directly in the browser — no account or login required.

It guides educators through each element step by step, combines them into a complete prompt, and scores the result against quality criteria so they can see what makes a prompt effective before pasting it into ChatGPT, Claude, Gemini, or any other LLM.

## Features

- **Build tab** — fill in each PARTS element with guided placeholders and examples, watch the combined prompt assemble in real time, copy to clipboard with one click
- **Evaluate tab** — automatic quality scoring (0–100) that checks whether each element is present, specific, and well-formed; highlights what to improve
- **Reflect tab** — structured reflection prompts to help educators think about how they would adapt the prompt for their own classroom context
- **Compare tab** — side-by-side comparison of two prompt versions to see how revisions improve output quality
- **Examples tab** — ready-made PARTS prompts for common K-12 EFL scenarios (reading, writing, listening, speaking) that teachers can load and modify
- Progress tracker showing completion across all five PARTS elements
- Fully responsive for laptop, tablet, and projected display
- Single self-contained HTML file with no external dependencies

## PARTS Framework

| Element | Question | Example |
|---------|----------|---------|
| **P** — Persona | Who should the AI act as? | "Act as an experienced EFL teacher..." |
| **A** — Aim | What is the purpose or goal? | "...to develop reading comprehension skills..." |
| **R** — Recipients | Who are your students? | "...for B1-level Turkish high school students aged 15–17 with L1 Turkish..." |
| **T** — Theme | What is the topic? | "...on the topic of climate change..." |
| **S** — Structure | What should the output look like? | "...as a 45-min lesson plan with 5 stages..." |

## How to Use

1. Open `index.html` in any modern browser (or visit the GitHub Pages URL)
2. Start on the **Build** tab — fill in each PARTS element using the guided fields
3. Copy the combined prompt and paste it into your preferred AI tool
4. Switch to the **Evaluate** tab to see your prompt's quality score and areas for improvement
5. Use the **Compare** tab to test how revisions change the output
