# PARTS Practice Tool

An interactive prompt-writing tool for educators based on the PARTS model.

**Live Tool:** https://kel03tr-ai.github.io/PARTS-Model/PARTS_Practice_Tool.html

## What is PARTS?
PARTS is a structured framework that helps teachers write effective AI prompts for classroom use. Each letter guides a key component of a well-formed prompt.

## How to Use
Open the live link above in any browser — no installation needed. Designed for use in professional development sessions as part of the AI in English Teaching Program (AIETP).
